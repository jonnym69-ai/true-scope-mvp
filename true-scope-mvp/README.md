## How to Use

1. **Describe your game idea** in the textarea (e.g., "cozy 2D farming game")
2. **Select your tool** (Unity, Godot, Unreal, etc.)
3. **Click "Generate Plan"** for a detailed roadmap
4. **Or click "Give me an idea"** for inspiration
